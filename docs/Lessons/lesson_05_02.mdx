# 5.2. Створення фінансового плану та документів

## Фінансові документи та розрахунки
### Видатки
У компанії є 2 типи видатків - постійні та змінні.
*Постійні видатки* - це ті, які не залежать від обсягу випуску (оренда приміщення, управлінські видатки, тощо).  
*Змінні видатки* - це ті, які залежать від обсягу випуску (сировина, оплата праці, вартість постачання, тощо).  
  
### Послідовність обрахунку  

Для того, щоб вести діяльність потрібні постійні видатки. Їх необхідно мінімізувати: працювати з дому або орендувати мінімальне приміщення, проте без видатків починати діяльність неможливо.  
  
Тому потрібно виписати всі видатки, які для ведення діяльності. При цьому в них варто вписати і власний час, обрахований по вартості вашої години.  
  
Після такого обрахунку ви отримаєте суму, яку має споживати ваш бізнес щомісяця.  

### Змінні витрати  
*Змінні витрати* - це витрати, які залежать від обсягу випускаємої продукції.  
Для їх обрахунку можна описати процес створення однієї одиниці продукції.   
В цьому обрахунку необхідно врахувати: витрати на створення одиниці продукції або собівартість; вартість залучення клієнта на 1 одиницю продукції; ціна продажу.  
На основі цих даних можна обрахувати валовий прибуток (Gross Profit)  

### Точка беззбитковості
Точкою беззбитковості називають такий обсяг продажів, в якій загальний обсяг валового прибутку дорівнює постійним витратам.  
Тобто це кількість одиниць товару, яку потрібно продати, щоб окупити всі постійні витрати, яких вимагає ведення підприємницької діяльності.  
  
### 3 ключових фінансових звіта
В фінансах є 3 ключових фінансових звіта:
* Звіт з грошового потоку (в ньому фіксується всі рухи грошових коштів)
* Звіт з доходів та розходів (в ньому фіксується весь виторг та всі видатки в цілому)
* Звіт з балансу (в ньому фіксується вся власність компанії)
