# Модуль 5. Фінанси та облік для малого бізнесу
Загальний опис: Соціальний капітал та продажі. Формуєте карту контактів. Готуєте скрипт продажів та опрацьовуєте його в групі, на знайомих та на потенційних клієнтах. Готуєте рекламні матеріали  та запускаєте перший тестпост у соцмережах та інших ресурсах, за відгуками отримуєте доказ, що ідея працює. На основі розуміння того, скільки потребує ресурсів підготовка рекламних матеріалів - вносити корективи до фінансового плану та уточнюєте механізми роботи із фінансами. Знайомитесь із методами організації процесів ( планування, контроль задач - управлінський облік)


Бізнес-фреймворки: Agile, SCRUM, беклог, Діагрма Гантта, Unit Economics, CashFlow, P&L, Balance Statement.
Засоби: програми для бухгалтерського обліку
Знання: методи планування, методи оцінки ризиків
Артефакти: 
9. Цифровий робочий зошит 1 (персональний). Карта контактів.
10. Рекламні матеріали. а. Тест-план публікації (візуал + CTA + метрики). б. Звіт про запуск тестпосту (1 сторінка, з 3 інсайтами та 3 рекомендаціями).
11. Фінансові документи:  CashFlow, P&L, Balance Statement
