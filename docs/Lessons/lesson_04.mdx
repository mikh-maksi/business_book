# Модуль 4. Продукт і ціннісна пропозиція
Загальний опис: Розуміння клієнта. Визначаєте додаткові сегменти цільової аудиторії, рекламні канали для кожного сегменту та формулюєте вимоги для матеріалів для цих каналів. Проводите опитування, що перевіряє сформульовані припущення щодо сегментації цільової аудиторії.

Розділ 5. Ваші плани продажів і маркетингу
Бізнес-фреймворки: Sales Funnell, AIDA, MVP, скрипт продажів, CAC.
Засоби: посадкова сторінка, комунікація в месенджерах.
Знання: основи продажів та маркетингу, цінова та нецінова конкуренція
Приклади: креативи (банери) аналогічних продуктів
