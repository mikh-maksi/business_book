# Модуль 7. Правові аспекти ведення бізнесу та підтвердження бізнес-ідеї.
Загальний опис: Валідація мінімальної ідеї (MVP). Тестуєте продукт «у полях», збираєте відгуки та коригуєте бізнес-модель на основі реальних даних. Надаєте тестові послуги та отримаєте 3 відгуки на них. Формуєте лист відповідей на запитання та всі супроводжувальні матеріали.


Типові договори
Засоби: сервіси для накладання електронного цифрового підпису («Вчасно»), типові договори; он-лайн засоби для державної реєстрації.
Знання: основи законодавства, що регулює підприємницьку діяльність

Артефакти:
	13. План тестової кампанії (кому і як надаємо тектовий послугу та продукт, як отримуємо відгук). 
	14. Q&A-лист (8–10 питань з аргументованими відповідями). 
15. Звіт тестування з ключовими метриками та висновками. 
	17. Оновлення електронного цифрового зошиту (бізнес). Оновлені BMC і фінплан з урахуванням результатів тесту.
18. Адаптація типових договорів.
