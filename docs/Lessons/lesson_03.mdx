# Модуль 3. Бізнес-модель

Загальний опис: Проєктування бізнесу: формуєте цифровий зошит 2 (бізнес) на основі Business Model Canvas (BMC), сценарію розвитку бізнесу, CJM, юніт-економіки. Робите перший варіант дашборд.

Розділ 6. Бізнес та Бізнес-ідея

Бізнес-фреймворки: Business Model Canvas, Бізнес ДНК, SMART-goal;
Знання: Мікроекономічна модель, структурна модель (Business Model Canvas), оцінка за кожним юнітом (Юніт-економіка).
Приклади: шаблони рекламних повідомлень
Практична дія: підготовка найпростіших рекламних повідомлень (з нуля), створення/адаптація з заготовки графічної реклами (банеру).
