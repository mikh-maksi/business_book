# Модуль 2. Підприємець та ринок, клієнт, попит.

Загальний опис: Внутрішня готовність. Створюєте персональну карту ролей та мотиваційний план на 3 місяці, щоб не «перегоріти» на старті. З'єднуєте бажаний напрямок діяльності (на основі власних компетенцій та схильностей) та існуючий попит. Робите зв'язок ключового сегменту цільової аудиторії із продуктом (Value Proposition Canvas).

Розділ 3. Підтримка мотивації підприємця.
Розділ 4. Цільова аудиторія. Ринок та конкуренція

Бізнес-фреймворки: Value Proposition Canvas, Аватар клієнта, Перелік конкурентів, презентація, що продає; УТП; SWOT-аналіз для оцінки ринку.
Засоби: графічний банер, форма опитування, презентація, скрінкаст
Знання: Особливості прийняття рішень, AIDA, рекламні канали; закон попиту і пропозиції; Типи ринків; Цінова та нецінова конкуренція;
Приклади: шаблони рекламних повідомлень
Практична дія: публікація в соціальних мережах, організація опитування

Артефакти:   
3. Цифровий робочий зошит 1 (розширення) а. Персональна карта ролей (візуальна). б. Мотиваційний план (3 місяців).  
4. Оцінка зовнішнього середовища (розширення). а. Портрет цільового клієнта (аватар клієнта) б. Канва ціннісної пропозиції (Value Proposition Canvas) в. Звіт з опитування (підтвердження ціннісної пропозиції в групи, в оточуючих, в представників цільової аудиторії). г. Приклади рекламних повідомлень для компанії-взірця
