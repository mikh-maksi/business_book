# DashBoard
Для того, щоб контролювати досягнення мети необхідно приймати операційні рішення. Для прийняття рішень необхідні дані. Дашборд - це таблиця, в яку вносяться дані, щодо діяльності.
В дашборді важливо контролювати як основні показники (чисельні показники визначеної мети), так і випереджувальні показники (вони впливають на показники результати, але можна відслідкувати негативні тенденції ще до того, як буде отриманий кінцевий результат).
Наявність дашборду, визначених показників, їх тижневий та місячний план дозволяють реагувати на наявну невідповідність одразу, як тільки її буде виявлено.
